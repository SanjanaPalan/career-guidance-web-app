# Career Guidance Web Application

## Overview

Career Guidance Web Application is a web-based platform that helps students explore suitable career paths based on their interests.

The application provides:

- Career recommendations
- Learning roadmaps
- Required skills
- Expected salary information
- Learning resources

## Features

- Career selection interface
- Learning roadmap generation
- Skills recommendation
- Salary insights
- Resource links for learning
- Responsive and user-friendly UI

## Installation

1. Clone the repository

```bash
git clone https://github.com/SanjanaPalan/career-guidance-web-app.git
```

2. Navigate to project folder

```bash
cd career-guidance-web-app
```

3. Install Flask

```bash
pip install flask
```

4. Run the application

```bash
py app.py
```

5. Open browser

```text
http://127.0.0.1:5000
```
